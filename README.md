## 🚀 Cara Menjalankan Project

1. **Clone repository ini**
   ```sh
   git clone https://github.com/JonathanZefanya/Alkitab-App.git
   cd alkitab-app
   ```

2. **Install dependencies**
   ```sh
   flutter pub get
   ```

3. **Jalankan aplikasi**
   ```sh
   flutter run
   ```
